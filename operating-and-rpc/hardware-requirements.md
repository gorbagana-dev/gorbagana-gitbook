# Minimum Hardware Requirements

To run a reliable RPC node for the Gorbagana network, ensure your system meets the following minimum specifications:

* **RAM**: Minimum 64GB (Recommended: 64GB for optimal performance)
* **Storage**: Minimum 1TB SSD (Recommended: 2TB+ NVMe SSD for faster I/O operations)
* **CPU**: Multi-core processor (Recommended: 8+ cores for better concurrent request handling)
* **Network**: Stable internet connection with high bandwidth (Recommended: 50+ Mbps upload/download)
* **Operating System**: Linux-based system (Ubuntu 20.04 LTS or newer recommended)

**Note**: These requirements ensure your RPC node can handle network synchronization, transaction processing, and serve multiple concurrent requests efficiently. Higher specifications will provide better performance and reliability.
